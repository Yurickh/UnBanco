Unidade de Testes
==============================================================================

	Para compilar e rodar a unidade de testes, basta usar o makefile que já está
previamente instalado com o UnBanco.

Linux:
	$ make teste
	$ ./teste

Windows:
	Abra o arquivo TestUnit.cpp com sua IDE e a compile. Seja feliz.
